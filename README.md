# 🌤️ Weather Now App

A modern weather application built using **HTML, CSS, and JavaScript**, powered by the Open-Meteo API.  
It provides real-time weather data, hourly forecasts, and daily forecasts with a clean UI inspired by Frontend Mentor design.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Switch between Metric and Imperial units
- 📅 7-day weather forecast
- ⏰ Next 8-hour hourly forecast (starting from current time)
- 💧 Humidity, Wind Speed, and Precipitation details
- 🎨 Modern UI with icons and responsive layout
- ⚡ Fast and optimized API handling
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6 Modules)
- Open-Meteo API
